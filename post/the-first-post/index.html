Hello World!

I've started this new site for a few reasons. First, I wanted a place to put short posts outside of my main blog. Second, I wanted to make use of Github pages and git. This page has said nothing but "Hello World!" for a little too long. Maintaining this site gives me a persistent reason to keep using git. Finally, I wanted get back to writing and designing HTML. Random Bits is an entirely static site--as all Github pages are.
